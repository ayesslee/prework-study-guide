Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user
  • w3schools.com for functions
  • The "link" element creates connection to external files
  • The "rel" attribute specifies relationship between current and linked source
  • The "href" attribute specifies the URL of the external resource

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element.
  • DRY (Do not Repeat Yourself)- a coding principle that shares pieces of code to reduce the number of lines in code
  • the "img" selector assigns property values to any image in the markup
  • the "block" value assigns things to be started on a new line

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull: fetch and download content from a remote repository and immediately update the local repository to match that content
  • git rebase: allows you to rewrite commits from one Git branch to another branch
